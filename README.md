# Yash Garg | Portfolio

Welcome to my personal portfolio website — a modern, responsive, and interactive space that showcases my journey as a developer, competitive programmer, and problem solver.

This portfolio is designed not just to look good, but to feel alive. It combines clean UI, smooth animations, a custom AI terminal, and a polished project showcase to create a strong first impression for recruiters, collaborators, and visitors.

---

## 🌐 Live Portfolio

**Frontend:** Deployed on Vercel  
**Backend:** Deployed on Render  
**AI Terminal:** Powered by Google Gemini API

---

## ✨ Overview

This portfolio is built to reflect my growth, skills, and interests in:

- **Full Stack Development**
- **Competitive Programming**
- **AI / ML Exploration**
- **Project Building**
- **Clean UI / UX Design**

It includes a mix of personal branding, technical highlights, and interactive elements so that anyone visiting the site can quickly understand who I am, what I do, and what I am building.
